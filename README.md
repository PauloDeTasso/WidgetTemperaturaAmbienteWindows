# 🌡️ WeatherWidget

Um widget simples e leve pra mostrar a temperatura atual da sua cidade na tela do Windows. Feito em **C# Windows Forms** com **.NET**, usando a API gratuita do **Open-Meteo**.  

> ⚡ Funciona com **hotkey F10** pra mostrar/esconder, e quando escondido ele para de atualizar pra não gastar processamento.

---

## 🛠️ Funcionalidades

- Mostra a **temperatura atual** com uma casa decimal (`0,0 °C`)  
- **Transparente** e sempre no topo da tela  
- **Hotkey global F10**: mostrar/esconder o widget  
- Atualiza automaticamente **a cada 30 minutos**  
- Usa **API gratuita e sem limite de chave** (`open-meteo.com`)  
- Quando escondido, o **timer para**, economizando recursos do PC  

---

## 💻 Requisitos

- Windows 10/11  
- .NET 6 ou superior  
- Visual Studio 2022 ou compatível
